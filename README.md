# InternProject
Spring Boot API test

1. install lombok.jar
2. import InternProject
3. Run on Server
4. test API


## Test API URI
[Get] : http://localhost:8080/api/test/user/{loginId}

[Get] : http://localhost:8080/api/test/user


## Version
* Java 8
* Spring Boot 2.5.3


## Dependencies
* Spring Boot DevTools
* Lombok
* Spring Data JPA
* MySQL Driver
* Spring Sercurity
* Spring Web
